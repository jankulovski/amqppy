amqppy
======
AMQP simplified client for Python

|Status| |Coverage|

Introduction
------------
amqppy is a very simplified AMQP client stacked over `Pika <https://github.com/pika/pika>`_. It has been tested only with `RabbitMQ <https://www.rabbitmq.com>`_
